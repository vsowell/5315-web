What is Docker

Docker is an open source software that provides containers.

These containers are like mini-storage systems that can run

filesystems in an enclosed environment independent of other

systems you may have installed on your own computer,

virtual desktop, or other medium.

Source:

  • https://en.wikipedia.org/wiki/Docker_(software)

What is Nginx

Nginx is an open-source webserver. Using HTTP, it processes requests.
It was created by Igor Sysoev and is a direct competitor of Apache.
The main benefits of Nginx over Apache are the ability to multitask
and run multiple processes at once and run different jobs during each process,
making it more efficient and requiring less memory to do so.
